Overview : 
This project focuses on analyzing an e-commerce dataset by building a complete data pipeline—from raw 
CSV files to structured SQL tables—and extracting meaningful business insights using SQL queries.

Objectives : 
1. Convert raw CSV data into a relational database
2. Perform data cleaning and transformation using Python
3. Analyze sales performance using SQL
4. Identify key business insights such as top-selling categories and revenue trends

Tech Stack : 
1. Python (Pandas, SQL connectors)
2. MySQL
3. SQL (Joins, Aggregations, Subqueries, CTEs)
4. Jupyter Notebook 

Workflow : 
1. Imported raw CSV datasets into Python
2. Transformed and loaded data into MySQL tables
3. Created relational schema across multiple tables (products, orders, payments)
4. Wrote SQL queries to analyze sales and customer behavior
5. Extracted insights such as top-performing product categories

Key Analysis Performed : 
1. Total sales by product category
2. Identification of highest revenue-generating category
3. Order and payment analysis using JOIN operations
4. Aggregation using SUM, GROUP BY, and ORDER BY
